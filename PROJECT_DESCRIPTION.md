# Project Description

## Short Description

AI-driven threat modeling and attack simulation framework for resilient cyber-physical systems, integrating STRIDE, PASTA, LSTM-based adversarial detection, dynamic risk analysis, and MITRE ATT&CK for ICS contextualization.

## Extended Description

This project develops a research-oriented cybersecurity workflow for industrial cyber-physical systems using the SWaT environment as a case study. It combines structured threat modeling, temporal machine learning, dynamic risk reasoning, comparative framework analysis, and ontology-backed explainability into a single end-to-end pipeline.

The work is designed to move from design-time threat identification to runtime attack detection and finally to analyst-facing interpretation, making it suitable for academic presentation, technical demonstration, and research-style documentation.

## Core Focus Areas

- AI-based threat modeling
- adversarial intelligence for CPS security
- learning-aware risk analysis
- attack simulation and resilience evaluation
- MITRE ATT&CK for ICS integration
- ontology-supported explainability
